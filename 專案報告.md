## 專案概述

本專案是一個基於 Qt 框架開發的圖片處理應用程式，主要實現了以下核心功能：

1. **拖移選取放大功能**：使用者可以在主視窗中用滑鼠拖移選取圖片的任意區域，選取完成後該區域會自動放大2倍並在新視窗中顯示。

2. **畫筆繪圖功能**：在放大視窗中，使用者可以使用多種顏色的畫筆在圖片上繪圖或標記。

3. **圖片儲存功能**：提供另存新檔功能，可將編輯後的圖片儲存為 PNG、JPEG 或 BMP 格式。

### 專案目標

- 提供直覺的圖片區域放大操作介面
- 支援在放大圖片上進行標記和註解
- 簡化的使用者介面，專注於核心功能
- 穩定的程式執行，無記憶體洩漏

```cpp
//---------------------------------------------------------
// 選取覆蓋層類別，用於在圖片上方顯示選取框
class SelectionOverlay : public QWidget
{
public:
    explicit SelectionOverlay(QWidget *parent = nullptr)
        : QWidget(parent)
    {
        setAttribute(Qt::WA_TransparentForMouseEvents);
        setAttribute(Qt::WA_TranslucentBackground);
    }
    
    void setSelectionRect(const QRect &rect)
    {
        selectionRect = rect;
        update();
    }
    
protected:
    void paintEvent(QPaintEvent *event) override
    {
        Q_UNUSED(event);
        if (!selectionRect.isNull())
        {
            QPainter painter(this);
            QPen pen(Qt::blue, 2, Qt::DashLine);
            painter.setPen(pen);
            painter.drawRect(selectionRect);
        }
    }
    
private:
    QRect selectionRect;
};
//---------------------------------------------------------------
```

#### 2. 滑鼠事件處理（mainwindow.cpp）

```cpp
//---------------------------------------------------------
// 滑鼠按下事件 - 主視窗開始選取，放大視窗開始繪圖
void MainWindow::mousePressEvent(QMouseEvent *event)
{
    if (event->button() == Qt::LeftButton && imgLabel->pixmap() && !imgLabel->pixmap()->isNull())
    {
        if (!isZoomedWindow) // 主視窗：選取功能
        {
            selectionStart = event->pos();
            isDragging = true;
        }
        else // 放大視窗：繪圖功能
        {
            isDrawing = true;
            lastDrawPoint = event->pos();
        }
    }
}
//---------------------------------------------------------------

//---------------------------------------------------------
// 滑鼠移動事件 - 主視窗更新選取框，放大視窗繪圖
void MainWindow::mouseMoveEvent(QMouseEvent *event)
{
    if (!isZoomedWindow && isDragging) // 主視窗：更新選取框
    {
        selectionEnd = event->pos();
        // 更新覆蓋層顯示
        if (overlay)
        {
            QPoint centralStart = imgLabel->mapFrom(this, selectionStart);
            QPoint centralEnd = imgLabel->mapFrom(this, selectionEnd);
            QRect rect = QRect(centralStart, centralEnd).normalized();
            overlay->setSelectionRect(rect);
        }
    }
    else if (isZoomedWindow && isDrawing) // 放大視窗：繪圖
    {
        QPoint currentPoint = event->pos();
        
        // 在圖片上繪圖
        QPixmap pixmap = *imgLabel->pixmap();
        QPainter painter(&pixmap);
        painter.setPen(QPen(penColor, penWidth, Qt::SolidLine, Qt::RoundCap, Qt::RoundJoin));
        
        // 將視窗座標轉換為圖片座標
        QPoint imgStart = imgLabel->mapFrom(this, lastDrawPoint);
        QPoint imgEnd = imgLabel->mapFrom(this, currentPoint);
        
        painter.drawLine(imgStart, imgEnd);
        imgLabel->setPixmap(pixmap);
        
        lastDrawPoint = currentPoint;
    }
}
//---------------------------------------------------------------
```

#### 3. 放大功能實作（mainwindow.cpp）

```cpp
//---------------------------------------------------------
// 滑鼠釋放事件 - 主視窗執行放大，放大視窗結束繪圖
void MainWindow::mouseReleaseEvent(QMouseEvent *event)
{
    if (event->button() == Qt::LeftButton)
    {
        if (!isZoomedWindow && isDragging) // 主視窗：執行放大
        {
            isDragging = false;
            
            // 計算選取區域
            QRect rect = QRect(selectionStart, selectionEnd).normalized();
            
            // 轉換為圖片座標
            QPoint imgStart = imgLabel->mapFrom(this, rect.topLeft());
            QPoint imgEnd = imgLabel->mapFrom(this, rect.bottomRight());
            QRect imgRect = QRect(imgStart, imgEnd).normalized();
            
            // 裁切並放大
            QPixmap originalPixmap = *imgLabel->pixmap();
            QPixmap cropped = originalPixmap.copy(imgRect);
            QPixmap zoomed = cropped.scaled(cropped.width() * 2, cropped.height() * 2);
            
            // 建立放大視窗
            MainWindow *zoomWindow = new MainWindow(nullptr, true);
            zoomWindow->setAttribute(Qt::WA_DeleteOnClose);
            zoomWindow->setWindowTitle("放大視窗 - 2x");
            zoomWindow->imgLabel->setPixmap(zoomed);
            zoomWindow->show();
            
            // 清除選取框
            if (overlay)
            {
                overlay->setSelectionRect(QRect());
            }
        }
        else if (isZoomedWindow) // 放大視窗：結束繪圖
        {
            isDrawing = false;
        }
    }
}
//---------------------------------------------------------------
```

## 實作細節

### 1. 選取框覆蓋層技術

**問題**：初始實作中，選取框顯示在圖片下方而非上方。

**解決方案**：
- 建立 `SelectionOverlay` 類別繼承自 `QWidget`
- 設定 `Qt::WA_TransparentForMouseEvents` 屬性使滑鼠事件穿透
- 設定 `Qt::WA_TranslucentBackground` 屬性使背景透明
- 將 overlay 設為 centralWidget 的子元件，確保顯示在圖片上方

**技術要點**：
```cpp
// 座標轉換：從 MainWindow 座標轉換為 imgLabel 座標
QPoint centralStart = imgLabel->mapFrom(this, selectionStart);
QPoint centralEnd = imgLabel->mapFrom(this, selectionEnd);
```

### 2. 雙視窗架構

**設計理念**：使用同一個 `MainWindow` 類別，透過 `isZoomedWindow` 標記區分主視窗和放大視窗。

**優點**：
- 程式碼重用
- 維護簡單
- 記憶體效率高

**實作方式**：
```cpp
// 建構函式根據參數建立不同功能的視窗
MainWindow::MainWindow(QWidget *parent, bool isZoomedWindow)
    : QMainWindow(parent)
    , isZoomedWindow(isZoomedWindow)
{
    // 初始化
    setupUi(this);
    
    if (isZoomedWindow)
    {
        // 放大視窗：簡化UI，啟用繪圖功能
        createZoomedWindowActions();
        createZoomedWindowMenus();
    }
    else
    {
        // 主視窗：完整UI，啟用選取功能
        createActions();
        createMenus();
        createToolBars();
        
        // 建立選取覆蓋層
        overlay = new SelectionOverlay(centralWidget());
    }
}
```

### 3. 繪圖功能實作

**繪圖流程**：
1. 使用者按下滑鼠左鍵 → 記錄起始點，設定 `isDrawing = true`
2. 使用者拖移滑鼠 → 從上一點畫線到當前點
3. 使用者釋放滑鼠 → 設定 `isDrawing = false`

**關鍵技術**：

## 問題與解決方案

### 問題 1：選取框顯示位置錯誤

**現象**：選取框顯示在圖片下方，而非覆蓋在圖片上方。

**原因分析**：
- 直接在 MainWindow 上繪製，Z-order 低於 imgLabel
- 座標系統未正確轉換

**解決方案**：
1. 建立獨立的 `SelectionOverlay` 類別
2. 將其設為 centralWidget 的子元件
3. 設定透明和滑鼠穿透屬性
4. 正確轉換座標系統

**提交記錄**：
```
commit 7455ac8
Fix selection rectangle positioning - overlay image properly
```

### 問題 2：連結器錯誤

**現象**：
```
error: ld returned 1 exit status
```

**原因分析**：
- `SelectionOverlay` 類別在標頭檔中定義但使用了 `Q_OBJECT` 巨集
- MOC (Meta-Object Compiler) 無法正確處理

**解決方案**：
1. 移除 `SelectionOverlay` 類別中的 `Q_OBJECT` 巨集
2. 該類別不需要 signal/slot 功能，不需要 MOC 處理
3. 執行完整的清理和重建

**提交記錄**：
```
commit 187ce12
Fix linker error by removing Q_OBJECT from SelectionOverlay class
```

### 問題 3：編譯器警告

**現象**：
```
warning: unused parameter 'event'
```

**原因分析**：
- `paintEvent` 函式參數 `event` 未使用

**解決方案**：
```cpp
void paintEvent(QPaintEvent *event) override
{
    Q_UNUSED(event);  // 明確標記未使用
    // ...
}
```

**提交記錄**：
```
commit 4c22542
Add Q_UNUSED to suppress compiler warning
```

### 問題 4：快速鍵衝突

**現象**：「另存新檔」和「縮小」功能的加速鍵都是 Alt+S。

**原因分析**：
- `tr("另存新檔(&S)")` 使用 &S
- `tr("縮小(&S)")` 也使用 &S

**解決方案**：
```cpp
// 將另存新檔的加速鍵改為 &A
saveAsAction = new QAction(tr("另存新檔(&A)"), this);
```

**提交記錄**：
```
commit f8f6f34
Fix accelerator key conflict - change Save As from &S to &A
```

### 問題 5：未初始化指標

**現象**：潛在的未定義行為風險。

**原因分析**：
- `saveAsAction` 指標在某些情況下未初始化就使用

**解決方案**：
```cpp
MainWindow::MainWindow(...)
{
    // 明確初始化為 nullptr
    saveAsAction = nullptr;
    overlay = nullptr;
}
```

**提交記錄**：
```

Initialize saveAsAction to nullptr
```
